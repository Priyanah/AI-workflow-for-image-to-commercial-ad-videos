# AI-workflow-for-image-to-commercial-ad-video

- I have developed an image-to-video generation system. 
- In this workflow, you just have to upload the image, any kind of image like watches, sunglasses, perfumes, etc., and also jewelry designs.
- Once the image is uploaded, the workflow processes it and delivers a commercial ad video that can be used to showcase the product. And this workflow can be used for content creation
- I have Develped this using N8N google pro models, code nodes, image dataset, https nodes, webhook nodes.
